DXVK
====

The following shared libraries shipped under Pulsar/Libraries/ are part of
DXVK (Direct3D 11 -> Vulkan translation layer), built from upstream sources
at https://github.com/doitsujin/dxvk (version v2.7.1, per the embedded
version string in libdxvk_d3d11.so):

    libdxvk_d3d11.so   (SONAME: libdxvk_d3d11.so.0)
    libdxvk_dxgi.so    (SONAME: libdxvk_dxgi.so.0)
